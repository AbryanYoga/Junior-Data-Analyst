## 📌 1. Tentang Program PKL

Repositori ini merupakan dokumentasi kegiatan, catatan pembelajaran, serta kumpulan proyek yang saya kerjakan selama melaksanakan Praktik Kerja Lapangan (PKL) pada bidang **Data Analytics**.

Selama kegiatan PKL, saya mempelajari bagaimana proses pengolahan data dilakukan mulai dari tahap pengumpulan data, pembersihan data, analisis, hingga penyajian hasil analisis dalam bentuk visualisasi dan dashboard.

Selain meningkatkan kemampuan teknis, kegiatan ini juga membantu saya memahami budaya kerja industri seperti manajemen waktu, penyelesaian tugas berdasarkan target, dokumentasi pekerjaan, serta kolaborasi dalam tim.

## 🛠️ 2. Core Competencies & Tech Stack

Selama program PKL, saya mengembangkan beberapa kompetensi berikut:

### 💻 A. Kompetensi Teknis

* **🗃️ Data Handling & Preparation**
  Melakukan pembersihan data (*data cleaning*), menangani data kosong (*missing values*), menghapus data duplikat, serta menyiapkan dataset agar siap dianalisis.

* **🔍 SQL for Data Analysis**
  Menulis query SQL untuk melakukan ekstraksi data, pengolahan data, penggabungan tabel menggunakan *JOIN*, dan analisis menggunakan fungsi agregasi.

* **📊 Exploratory Data Analysis (EDA)**
  Mengeksplorasi data untuk menemukan pola, tren, serta menghasilkan insight yang dapat digunakan sebagai dasar pengambilan keputusan.

* **📈 Data Visualization & Dashboard**
  Membuat visualisasi data dan dashboard sederhana untuk menyampaikan informasi kepada pengguna non-teknis.

### ⚙️ B. Kompetensi Industri

* **💡 Problem Solving & Data Thinking**
  Menganalisis permasalahan dan menerjemahkannya menjadi pertanyaan yang dapat dijawab menggunakan data.

* **📂 Documentation & Version Control**
  Mendokumentasikan pekerjaan secara terstruktur serta mengelola proyek menggunakan Git dan GitHub.

* **🗣️ Data Storytelling**
  Menyampaikan hasil analisis dalam bentuk laporan maupun presentasi agar lebih mudah dipahami.
